Proyectos

E-commerce de Bombones

Dulce Magnolia es un e-commerce realizado en PHP y MySQL. Este proyecto posee 4 niveles de usuarios. Invitado, el cual solo puede ver la página. Cliente el cual puede ejecutar compras. Administrador y SuperAdmin, los cuales tienen acceso al panel administrador, pueden ejecutar acciones CRUD sobre todos los productos. Todo el sitio interactúa con la base de datos y trae la información en tiempo real.

Presionando la vista previa podés visitar mi proyecto en línea.

Servicio Meteorológico

Utilizando la API de openweather, desarrolle esta aplicación. Te permite buscar por ciudad o país, y te muestra el clima de ese lugar, trae también temperatura máxima, mínima, humedad, sensación térmica y viento. También muestra el mapa del sitio buscado y el estado actual de las nubes en ese momento. Empleando el DOM también muestra la última búsqueda realizada.

Presionando la vista previa podés visitar mi Vercel.

Buscador de peliculas

LuFlix es una PWA alimentada por la API de omdbapi, en ella podés buscar películas y te aparece toda la información, además podés agregarla a favoritos y ver un tráiler. Usa Local Storage para guardar las películas en favoritos. También tiene un juego hecho en JS el cual aparece cuando se encuentra sin conexión.
